## :gear: How to Run:
First, clone the project:
```bash
git clone https://github.com/gabrielzj/cryptography_challenge
``` 

Then, install the dependencies:
```bash
pip install -r requirements.txt
```
Apply migrations:
```bash
python manage.py migrate
```

Run the server:
```bash
python manage.py runserver
```
